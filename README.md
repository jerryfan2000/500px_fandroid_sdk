500px-android-sdk
=================


Installation
------------

- clone the repository  
    <pre>git clone git://github.com/500px/500px-android-sdk</pre>
- add it as a dependency in your project  
    <pre>android.library.reference.1=../../../500px-android-sdk</pre>
